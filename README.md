🚀 SocialHub — Full Stack Social Media Web Application

SocialHub is a full-stack social media web application built to demonstrate real-world backend architecture, secure authentication & authorization, RESTful API design, and responsive UI development. The project focuses on implementing production-style user management and protected features using modern web technologies.

🔗 Live Demo

👉 https://socialhub-1-4qvg.onrender.com

JavaScript

Database

✨ Features

✅ User signup and login system

✅ Secure authentication using JWT

✅ Authorization with protected routes

✅ Password hashing with bcrypt

✅ MongoDB data storage with Mongoose schemas

✅ Create / Read / Update / Delete (CRUD) operations

✅ RESTful API architecture

✅ Express middleware for authentication & error handling

✅ Responsive UI built with Tailwind CSS

✅ Form validation and server-side checks

✅ Clean MVC-style project structure

⚙️ Installation & Setup (Local)
# clone repo
git clone https://github.com/Sumit93500/SocialHub.git

# go into folder
cd SocialHub

# install dependencies
npm install

# create .env file and add:
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

# run server
npm start
